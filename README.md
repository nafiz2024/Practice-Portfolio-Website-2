# Knowledge Vault

<img src="./Knowledge Vault.png">

## Overview
**Knowledge Vault** is a personal/project website that showcases plans, goals, and motivational content. The site includes sections for introduction, personal experiences, future plans, banners, and a subscription footer. It features a clean and modern design, built using **HTML, CSS, and JavaScript**, with Google Fonts and Font Awesome icons.

## Features
- Responsive and mobile-friendly layout
- Header section with motivational content and call-to-action button
- About section highlighting personal failures and lessons
- Plan section outlining goals and areas of focus
- Banner section with motivational messages and action button
- Footer section with contact information and newsletter subscription
- Use of icons to enhance UI elements

## Technologies Used
- HTML5
- CSS3
- Google Fonts (Montserrat, Open Sans, Work Sans)
- Font Awesome Icons
- Images for icons and visuals
